API Demo

Request Launchpad

Use this demo to create a photo request. Choose either Deeplink or API to start the flow. The demo generates a random customer_no (e.g. employee number, customer number, or process ID) to identify the user.

Deeplink: Use this if you want to display a link or button that starts the photo upload without creating an API request each time the page is shown.

API: Use this if the process is triggered by a backend event and you want Photo Collect to act as a status engine for pending requests.

After the photo is uploaded, the flow returns to this demo to fetch and display the exported photo.

Server-side handling

The Deeplink URL and the API request are sent via the server to keep the Deeplink secret and API-key hidden.

The server proxy is implemented with Slim PHP micro framewrok.

Check out the Github repo for the demo app source code.
